# 四角標音專案
本專案以XeLaTeX與Python實現，提供：
* 四角標音框架：在一個漢字大小的空間裡表達一個音節
* 豎排混合排版：將該音節方塊嵌入漢字竪排版文章
* 注音排版框架：在竪排文章裡為每一個字注音

## 內建四角標音示例：fcTailo
最開始，這只是一個紀念阿媽的小計畫。阿媽是一位虔誠的佛教徒，因此私以為若把手上留存的阿媽誦讀佛經的錄音檔記錄成書面標音檔案，是僅屬於我的紀念方式。
但在思考採用什麼標音系統的過程中，既然佛經一般是竪排版，我意識到現行諸多記錄工具頗有不便：
- 西文字母方案有三種做法：
  - 直接橫排標在右側：行距會忽大忽小，因為tshiong與a都是一個音節
  - 轉九十度標在右側：每次看羅馬字都要脖子轉九十度
  - 竪排字母標在右側：音節字母割裂開來極其難讀
因此不得不考慮天然支持竪排的東亞表音文字。不過，假名、諺文或注音方案都有得失。
- 假名：
  - 得：Unicode提供了聲調符號等書寫台語音的字符。
  - 失：日語音系太過簡單，導致可能需要四個假名才能表達一個音節。
- 諺文：
  - 得：諺文是設計為一方塊表達一音節，非常優雅。
  - 失：它沒有聲調支援，且今天不在預組字區段內的諺文動態組字比較不穩定。
- 注音：
  - 得：理論上音系支援充足。
  - 失：又想要維持華語預組韻母的格局，在韻母豐富的台語卻又不得不引入韻母分析，導致自相矛盾，如ang與ak是平等的韻母，前者一個符號後者兩個符號。
因此，我設計了一套每個音節固定用四個符號拼起來的框架，稱為`fcframe`。我也給出了一個引用該框架的示例，即`fcTailo`，它是一個基於注音符號、但完全不使用預組韻母（an、ang、am...）的系統，其符號表（`symbols.json`）與音系（`phonology.json`）可見於`json/Tailo/`目錄下。`fcTailo`實際上是利用這兩個`.json`檔案自動產生。
在main.tex引用`\usepackage{fcTailo}`後，您就可以使用以下巨集：
- `\SymbolTailo{p}% 這會顯示台羅p對應的符號ㄅ`
- `\fromTailo{e7}% 這會在竪排漢字混排台羅的ē對應的四角標音`
- `\Mark{台}{\SyllableTailo{tai5}}\Mark{灣}{\SyllableTailo[last]{uan5}}% 這會顯示右側帶有tai5-uan5四角標音的「台灣」，[last]表示這個字讀本調，四角符號的右下角會多一個記號。`
注意，如果需要有時標音有時不標音，行距可能忽大忽小，因為`\Mark`巨集是用TikZ寫的。所以，我更推薦全都不標音、僅混合排版或全都標音。
當然，手動標音很麻煩，因此推薦使用自動化工具。這將在後文敘述。

## 自定義四角標音
`frframe`只是一個框架，並不綁死在注音符號。您也可以自定義四角標音符號，這可能需要一些Python知識，因為枚舉音節空間往往需要數千行，交給自動化工具更好一些。自定義流程如下：
1. 在`json/`下新建一個檔案夾，例如`mysystem`。在`json/mysystem/`下建立`symbol.json`與`phonology.json`
2. 打開`json/mysystem/symbol.json`，定義您系統需要的符號：
  - 請參照`json/Tailo/symbol.json`的方式定義。
  - 一般Unicode字元請使用小寫u開頭，例如u3105表示ㄅ。
  - 組合用濁點可能在前或在後，視乎字型。我採用的是一點明體（[https://github.com/ichitenfont/I.Ming]），濁點在符號之前，如u3099u3105在一點明體表示帶濁點的ㄅ。如果您使用Noto Sans CJK，則這個序列應該是u3105u3099。
  - 由於使用LaTeX，我提供了「+」這個特殊符號，跑python後它會轉換成`\ooalign`巨集把前後兩個字疊起來。
  - 符號表要有聲調符號。
3. 打開`json/mysystem/symbol.json`，定義您系統的「不含聲調」音系
  - 請參照`json/Tailo/phonology.json`的方式定義。
  - 我不推薦寫代碼分析rime（韻母），因為處理閉元音及其對應滑音的邏輯會導致代碼很複雜且很難除錯。相較之下，即使韻母非常豐富的台語，也只需要100行左右就能枚舉完，還可以隨時回去除錯。
  - 韻母分析不一定要照glide、nucleus、coda分析，例如粵語就沒有韻母介音。
4. 請在`FourCornerTools/`下建立`mysystem.py`，參照`FourCornerTools/Tailo.py`的`FcMakerTailo`類別定義您的`FcMakerMysystem`類別。
  - 注意：您的類別要是`FourCornerTools/shared.py`定義的`FcMaker`的子類別。
  - 如果沒有特別的變調需求，您可以不實作`_sandhi`函式。詞首本調、其餘變調定義為`regs[0].sandhi = '[first]'`，詞末本調、其餘變調定義為`regs[-1].sandhi = '[last]'`（如範例的台語）。
  - 請您實作`syllable_lister()`函式，參照範例寫下枚舉`mysystem`所有「含聲調」音節的代碼。
5. 打開`main.py`，創建您類別的物件，執行其成員函式`.lister()`。
6. 檢查`fcmysystem.sty`是否正常產生，打開它看裡面是否有正確的符號巨集與四角標音巨集。此`.sty`應該會是一個數千行的檔案。
7. 若您能夠專門在`description/`檔案夾下寫一個`.tex`描述您的系統，或許會更有利於可視化。畢竟`.json`並不總是很直觀。

## 自動標音工具
`FcMaker.marker()`函式，可依照對應字典給原檔案標音並產生對應的`.tex`檔案，只要在`main.tex`裡引用這個標音檔案，就可以一鍵編譯產生注音的`.pdf`。流程如下
1. 進入`json/mysystem/`檔案夾，新增字典（如`mydict.json`）。它不需要收錄很多字，只需要包括您需要標音的檔案的所有字即可。
  - 請參照範例`json/Tailo/grandmadict.json`，如前所述，本計劃緣起紀念阿媽念佛，所以這個字典提供我自己擁有的錄音檔資料的詞目——讀音。
  - 每一個詞目可以是單字或詞。
  - 書寫使用的漢字目前僅包括基本平面，若有任何罕用字需求，請修改`FourCornerTools/shared.py`的`isCJK`函式內的正規表示法。同理，有其他標點符號需求可修改同一檔案內的`isPUNCT`函式。
2. 進入`rawtxt/`檔案夾，新增您需要標注的檔案的上層檔案夾，例如`myfolder`。再進入`myfolder/`，新增檔案，例如`to_be_marked.txt`。
  - 這是因為四角標音完全使用TikZ繪製，建議長文本分成多個檔案，用`main.tex`分別編譯每個小檔案並校對後修改名稱，然後用`\includepdf`巨集拼在一起，以便除錯。
3. 修改`main.py`，創建您類別的物件，執行其成員函式`.marker(folder_name='myfolder', dict_source='mydict', max_detect_length=4)`。
  - 注意傳入的字典**不要**含有副檔名。
  - 若您的字典含有更長的詞，調用`.marker`時記得指定`max_detect_length`。
  - 這個函式會訪問`rawtxt/`下所有的`.txt`檔案，並在`marked/myfolder/`檔案夾下產生對應的`.tex`檔案
  - 變調單位之間請使用半形空格。
  - 如果您沒有指定`FcMakerMysystem._sandhi`函式，則半形空格會被自動忽略。
4. 確認`marked/myfolder/`下正確產生`.tex`後，修改`main.tex`嘗試編譯。
如開頭所說，本計畫緣起紀念念佛的阿媽，所以提供的範例是兩部經書及阿媽讀經書的台語讀音，她的讀音與教育部辭典、台日大辭典等權威書目有細微不同。我在校對完後會把`main.py`利用命令列手動存到`result/`檔案夾下，但您也可以自己寫一個`.py`程式，自動調用命令列工具執行`xelatex`編譯並`mv`到您需要的檔案夾。
